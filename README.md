# NASC_094

Static course website for **NASC 094: Adventures in Science (Data Science focus)**.

## Contents

- `index.html`: landing page
- `syllabus.html`: full syllabus page
- `assets/styles.css`: shared site styles
- `.nojekyll`: disables Jekyll processing on GitHub Pages

## Publish on GitHub Pages

1. Create a GitHub repository named `NASC_094`.
2. Copy the contents of this folder into that repository.
3. Push to the `main` branch.
4. In GitHub, open `Settings -> Pages`.
5. Set the source to `Deploy from a branch`.
6. Choose `main` and the `/ (root)` folder.

Your site URL will be:

`https://feizhe.github.io/NASC_094/`

## Link from your personal site

Add a link on `https://feizhe.github.io/pages/courses.html` pointing to:

`https://feizhe.github.io/NASC_094/`

Example:

```html
<a href="https://feizhe.github.io/NASC_094/">NASC 094: Adventures in Science</a>
```
